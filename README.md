# 🛒 E-commerce SQL Analytics Project

## 📌 Objective
Analyze customer behavior, sales performance, and product trends using SQL.

---

## 🛠 Tools Used
- SQL (Joins, Aggregations, Window Functions)
- PostgreSQL / MySQL

---

## 📊 Key Analysis Performed

### 🔹 Basic Analysis
- Total customers
- Total revenue
- Average order value

### 🔹 Customer Insights
- Top customers by revenue
- Repeat customers

### 🔹 Sales Trends
- Running revenue over time
- Monthly trends

### 🔹 Advanced Analytics
- Customer ranking using window functions
- Revenue contribution %
- Category-wise revenue

---

## 📈 Key Insights
- A small group of customers contributes most of the revenue
- Repeat customers drive consistent sales
- Certain product categories dominate revenue

---

## 📂 Project Structure
- `schema.sql` → Table creation
- `data.sql` → Sample data
- `analysis.sql` → All queries
